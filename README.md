# AI-Driven Intrusion Detection and Prevention System (IDPS)

## Overview

This project implements an AI-driven Intrusion Detection and Prevention System using the CICIDS2017 dataset.

## Technologies

- Python
- Pandas
- Scikit-Learn
- Random Forest
- Isolation Forest
- Matplotlib
- Seaborn

## Dataset

CICIDS2017

Download:
https://www.unb.ca/cic/datasets/ids-2017.html

## Project Structure

- Data Preprocessing
- Random Forest Classification
- Isolation Forest Anomaly Detection
- Model Evaluation
- Alert Generation
- Simulated IP Blocking

## Results

Random Forest Accuracy:
99.73%
